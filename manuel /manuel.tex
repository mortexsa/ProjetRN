Lors de l'utilisation de l'application une page s'ouvre à afin de pouvoir faire ses manipulations. Un menu est à la portée : comme le montre la figure1.
\begin{center} 
			\includegraphics[height=244, width=300]{acceuil.png}
		\end{center}
figure1

\begin{itemize}
	\item Sélectionner (1)  un parmi d'autres réseaux de neurones déjà créés préalablement. En choisir un puis cliquer dessus, une page manipulation Réseau s'ouvrira figure 2. 
	\item Cliquer sur le bouton créer (2) pour créer un nouveau réseau de neurones. Une page création réseau de neurones s'ouvrira  figure3.
	\item cliquer sur le bouton Quitter (3) pour quitter l'application.
\end{itemize}

\begin{center} 
			\includegraphics[height=244, width=300]{manip.png}
		\end{center}
figure2

\begin{itemize}
	\item(1) Affichage des informations sur le réseau de neurones.
	\item cliquer sur le bouton Traitement (3) une page traitement d'images s'ouvrira : comme le montre la figure4.
	\item cliquer sur le bouton Apprentissage (4) lancera l'apprentissage du réseau de neurones, un deuxième clique sur le bouton arrêtera l'apprentissage.
	\item cliquer sur le bouton Retour (2) fait retour à la page d'accueil Figure 1. 
\end{itemize}

\begin{center} 
			\includegraphics[height=244, width=300]{creation.png}
		\end{center}
figure3

\begin{itemize}
	\item (1) remplir les champs avec les bons paramètres du réseau
	\item cliquer sur le bouton répertoire (2) pour choisir un répertoire d'images utile pour l'apprentissage.
	\item cliquer sur le bouton créer (3) pour valider les informations et se diriger vers la page manipulation réseau figure2.
	\item cliquer sur le bouton Retour(4) pour retourner à la page d'accueil figure1
\end{itemize}

\begin{center} 
			\includegraphics[height=244, width=300]{traitement.png}
		\end{center}
figure4

\begin{itemize}
	\item cliquer sur le bouton retour pour revenir à la page manipulation Réseau figure2.
	\item cliquer sur le bouton choisir fichier (1) pour pouvoir sélectionner l'image à analyser : comme le montre la figure ci-dessous.
\end{itemize}

\begin{center} 
			\includegraphics[height=244, width=300]{cap_fichier.png}

\begin{itemize}
	\item cliquer sur ouvrir pour valider le choix du fichier.
	\item cliquer sur annuler pour annuler et quitter la page.
	\item cliquer sur le bouton Lancer (3) ouvrira la page Top3 des résultats figure5. 

\end{itemize}

figure5

	\item cliquer sur le bouton ok (1) retour à la page traitement d'image figure4.
	\item cliquer sur le bouton quitter (2) pour quitter l'application.
\end{itemize}
